# Algorithm（算法）仓库

## 简介
探索经典算法，实现高效解题与优化。

## 目录结构
- **优选算法**: 优化问题的高效算法
- **贪心算法**: 贪心策略解决方案
- **动态规划**: 递推与记忆化搜索
- **递归与回溯**: 递归、搜索、回溯算法

## 使用方式
此仓库包含多个算法的C++实现，可以直接下载或克隆代码进行学习和应用。

## 适用人群
  - 学习 C++ 编程语言的初学者
  - 希望通过刷题提升算法能力的开发者
